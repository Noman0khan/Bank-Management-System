# 🏦 BUBT Banking System (C++)

A console-based Bank Management System built in **C++** as a group academic project. 
Demonstrates core **Data Structures** — Hash Table, Queue, and Stack.

---

## 📌 Features

- Create Bank Account using **Hash Table** (Linear Probing)
- Join service line using **Queue (Enqueue)**
- Leave from any position — **Middle Queue Removal**
- Serve next customer using **Queue (Dequeue)**
- Search account by ID using **Hash Table Search**
- View last action using **Stack (Top)**

---

## 🧠 Data Structures Used

| Feature | Data Structure |
|---|---|
| Account storage & lookup | Hash Table (Linear Probing) |
| Customer service line | Queue (Array-based) |
| Action history | Stack |
